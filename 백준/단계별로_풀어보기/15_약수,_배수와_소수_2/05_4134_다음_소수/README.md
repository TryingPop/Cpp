# C++17

## 난이도 : 실버 4

## 알고리즘 분류
  - 수학
  - 브루트포스 알고리즘
  - 정수론
  - 소수 판정

## 제한조건
  - 시간 제한 : 1초
  - 메모리 제한 : 128 MB

## 문제
  - 정수 n(0 ≤ n ≤ 4*109)가 주어졌을 때, n보다 크거나 같은 소수 중 가장 작은 소수 찾는 프로그램을 작성하시오.

## 입력
  - 첫째 줄에 테스트 케이스의 개수가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 n이 주어진다.

## 출력
  - 각각의 테스트 케이스에 대해서 n보다 크거나 같은 소수 중 가장 작은 소수를 한 줄에 하나씩 출력한다.
